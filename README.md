# Portfolio de Lénaïc Demont

## 🌐 Lien du site web
👉 [Accéder à mon site web GitHub Pages](https://lenaicdemontpro-boop.github.io)

## 🛠️ Technologies utilisées
* **HTML5** : Structuration du contenu de la page d'accueil.
* **CSS3** : Design responsive, gestion des couleurs (thème sombre/bleu) et de la mise en page.
* **Git & GitHub** : Gestion de version et hébergement via GitHub Pages.

## 📂 Contenu du dépôt
* `index.html` : Page d'accueil principale contenant ma présentation et mes projets.
* `_config.yml` : Fichier de configuration pour le thème et les métadonnées.
